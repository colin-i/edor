# EDitOR
Use autoconf automake libncurses-dev ; on armv7l, libunwind-dev
```sh
autoreconf -i
./configure
make install
```
Use *\-\-prefix=your_path* at *./configure* if needed (example: at Termux in Android).\
Uninstall command is *make uninstall*.\
*\-\-disable\-cpp* to set c rules.
